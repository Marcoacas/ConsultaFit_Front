Consulta Fit

e-commerce

Nuestro proyecto es un e-commerce desarrollado para la empresa Consulta Fit, la cual se especializa en la venta de ropa y artículos deportivos, así como planes de entrenamiento y consultoría nutricional. Ésta nace a partir de la pandemia, por la necesidad de realizar actividad física, desde la comodidad de tu casa.

Las tecnologías que usamos para la realización de este proyecto son:

Front End Back End
HTML Java
CSS Java Frameworks (Springboot)
CSS Frameworks (Bootstrap) MySQL
JavaScript
